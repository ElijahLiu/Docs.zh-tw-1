在本節中，您將建立主要應用程式頁面。 此頁面會比系統管理員 頁面上，更為複雜，所以我們會愈趨近它以幾個步驟。 過程中，您會看到一些更進階的解 Knockout.js 技術。 以下是頁面的基本的版面配置:

![](using-web-api-with-entity-framework-part-7/_static/image1.png)

- "Products"保存產品的陣列。

- [詳細資料] 保留順序的詳細資料，也就是陣列的項目 （產品與數量）
