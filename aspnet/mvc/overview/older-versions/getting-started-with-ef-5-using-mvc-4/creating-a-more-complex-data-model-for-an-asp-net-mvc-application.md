無法判定類型 'ContosoUniversity.Models.OfficeAssignment' 和 'ContosoUniversity.Models.Instructor' 之間的關聯的主要端點。 此關聯的主要端點必須使用明確設定關聯性 fluent 應用程式開發介面或資料註解。
