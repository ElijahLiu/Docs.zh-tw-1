## <a name="wrapup-and-review"></a>簡便和檢閱

本教學課程已引進新的行動裝置功能的 ASP.NET MVC 4 Developer Preview。 行動裝置的功能包括：

- 覆寫版面配置、 檢視和部分檢視，全域和個別的檢視能力。
- 控制版面配置，以及部分覆寫強制使用`RequireConsistentDisplayMode`屬性。

- 支援特定瀏覽器，例如 iPhone 瀏覽器支援。
