## <a name="overview"></a>總覽

`TextBoxWatermark` AJAX Control Toolkit 中的控制會擴充文字方塊中，使方塊內會顯示文字。 當使用者按一下此方塊時，它會清空。 如果使用者離開 [] 方塊中，而不需輸入文字，其中已預先的文字會重新出現。 這可能會與在相同頁面上，ASP.NET 驗證控制項發生衝突，但可能會解決這些問題。
