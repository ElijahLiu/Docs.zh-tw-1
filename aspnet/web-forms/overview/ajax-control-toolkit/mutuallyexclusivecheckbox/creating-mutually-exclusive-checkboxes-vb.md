## <a name="overview"></a>總覽

只有其中一個的一組選項可能會被選取時，通常用選項按鈕。 雖然會有缺點： 一旦選取一個選項按鈕群組中的，不可能取消選取所有選項按鈕。 核取方塊可以是未檢查在任何時間，但是不會互斥。 本教學課程提供了最佳的這兩種方法： 互斥的核取方塊。
